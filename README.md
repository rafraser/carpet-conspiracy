# Carpet Conspiracy

Carpet Conspiracy is a collection of web-scrapers for various fabric patterns.

## Running

Help:

```bash
python -m conspiracy -h
```

To run a single scraper:

```bash
python -m conspiracy astro
```

To run all scrapers:

```bash
python -m conspiracy all
```
